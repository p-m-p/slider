BoxSlider is a small library with no third party dependencies that provides a light-weight, responsive content carousel (slider) with
various slide transition effects for modern browsers. The core JavaScript library can be used standalone and there are React component
implementations for each slide effect. Web components are also available for each slide effect meaning the library can be used with
most front-end frameworks including Angular, Svelte and Vue.js.

## Should I use BoxSlider?

The carousel pattern gets a bad rap for being overused and often misused. Carousels still have a place in modern web design
when used correctly and they still get requested by clients. BoxSlider is a good choice if you need a simple solution and can
applied with best practices so that it does not negatively impact your users.
See [this guide](https://webflow.com/blog/carousel-slider-design-best-practices) for more information of the
design best practices.

## Key features

- **Performance**: BoxSlider is a small library with no dependencies. It is designed to be fast and efficient.
- **Accessibility**: BoxSlider is accessible. It is keyboard navigable and can be used with screen readers.
- **Customizable**: BoxSlider is highly customizable. You can change the layout, styling and other options to suit your needs.
- **Responsive**: BoxSlider is responsive. It will adapt to changes in screen size and orientation.
- **Modern**: BoxSlider is built with modern web technologies. It uses CSS animations and transitions for smooth performance.
