BoxSlider the light-weight content slider for modern web applications

Features

Multiple effects

Image and HTML slide effects with linear and 3D transitions

Adaptive

Adapts to changes in screen size or orientation without page jank

TypeScript support

Built for modern browsers and UI frameworks

Device ready

Works on all devices with support for swipe gesture transitions

Installation and usage

Install the BoxSlider package via NPM

npm install @boxslider/slider

Alternatively using Yarn 

yarn add @boxslider/slider

Import the BoxSlider library with the desired effect and instantiate the slider. See the examples for more details.

import { BoxSlider, CubeSlider } from '@boxslider/slider';

const slider = new BoxSlider(document.getElementById('cube-slider'), {
  effect: new CubeSlider({
    direction: 'horizontal'
  }),
  autoScroll: true,
  timeout: 5000
});
